== Description ==

Description: Freeware is a fast, clean, and modern-looking responsive Free WordPress Blog Theme. Minimal, elegant, lightweight, fast & mobile friendly layout with WooCommerce compatibility. This is a flexible template uses fresh and clean design.This theme is the best choice even for personal or professional websites. Get free Supports at https://themespiral.com/forums, View demos at https://demo.themespiral.com/freeware and documentation at https://docs.themespiral.com/freeware

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

= How to install Dummy Content ? =
 Please install One Click Demo Import plugins. You can install it after activating freeware theme. It is listed in recommended Plugins
 After plugin is activated, it asks you to upload  XML, WIE and  DAT dummy file
 Download it from https://themespiral.com/download/1369/
 Unzip freeware-dummy-content.zip file. You can find all XML, WIE and  DAT dummy file
 Upload manually and Click on Import demo data.

 Now all your files and settings has been imported. Now you just need to setup your menu.

 Setup Menu:
 In the Blog Dashboard, select Appearance > Menus.
 Under the Menu Settings, located at the bottom of your screen, select Primary/ Social Links
 Click save menu
 Screenshot: https://docs.themespiral.com/wp-content/uploads/2019/02/primary-menu.png
